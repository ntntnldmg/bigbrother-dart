# Flame Game Jam 2026

Source code for [Terrorist Threat](https://ntntnldmg.itch.io/terrorist-threat), our submission for the [5th Official Flame Game Jam](https://itch.io/jam/flame-game-jam-2026).

## Requirements

[Flutter](https://flutter.dev/)

## Usage

```cmd
git clone https://github.com/ntntnldmg/flame-game-jam-2026
cd flame-game-jam-2026
flutter run
```
